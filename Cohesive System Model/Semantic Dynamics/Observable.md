# Observable

An Observable is a probe, projection, measurement, or accessor that produces an [[Observation]] from [[State]].

Observables define how abstract state becomes visible to an [[Observer]]. Different observables may produce different observations from the same underlying state.

An observable is not itself the state. It is the access path, view, or measurement by which state is shaped into a usable [[Value|value]] with a declared [[Shape]].

A [[Query]] selects, invokes, or parameterizes an observable relative to an observer, boundary, authority, and consistency expectation.

Related concepts: [[State]], [[Value]], [[Shape]], [[Observation]], [[Query]], [[Observer]], [[Projections|Projection]].
